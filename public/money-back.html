Your Purchase is Protected

At Wrestle Swap, we want you to buy with confidence. That's why every eligible purchase is covered by our Money Back Guarantee. If an item doesn't arrive or isn't as described, we've got you covered.

‚úì You're covered when you:
  • Don't receive an item you ordered
  • Receive an item that is significantly different from the description
  • Receive a damaged or defective item
  • Return an item that doesn't match the listing

What's Covered

üì¶ Item Not Received

If your item doesn't arrive, you can request a refund. This includes situations where:

  • The tracking shows delivered but you didn't receive it
  • The item was lost in transit
  • The seller didn't ship within their stated handling time

üìã Item Not as Described

You're protected if the item you receive is materially different from the listing. This includes:

  • Wrong item was sent (e.g., you ordered a Large singlet but received a Medium)
  • Missing parts or accessories that were included in the description
  • Condition is significantly worse than described (e.g., listed as "Like New" but arrived heavily worn)
  • Item is damaged and damage wasn't disclosed in the listing
  • Item is a different brand, model, or version than listed

üíî Damaged or Defective Items

Items that arrive damaged or don't work as intended are covered, including:

  • Items damaged during shipping
  • Items with defects not mentioned in the listing
  • Items that don't function properly (e.g., zippers broken, tears in fabric)

What's Not Covered

‚úó The Money Back Guarantee does not cover:
  • Items picked up in person (local pickup transactions)
  • Buyer's remorse or change of mind
  • Minor differences in color due to screen variations
  • Items damaged after delivery
  • Normal wear and tear on used items when condition was accurately described
  • Items returned without seller authorization (unless item not as described)

üì¨ Return Shipping

Who Pays for Return Shipping?

  • Item not as described or damaged: Seller pays return shipping
  • Item not received: No return shipping needed (item never arrived)
  • Buyer's remorse: Buyer pays return shipping (if seller accepts returns)

When the seller is responsible for return shipping, we'll provide a prepaid return label for you to use.

  • Provide valid tracking information
  • Package items securely to prevent damage
  • Respond to buyer messages within 48 hours
  • Accept returns when items don't match the description
